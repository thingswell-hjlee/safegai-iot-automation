# FEAT-004: M01 Hybrid App Mock

Status: in_progress

## Description
Create React/TypeScript frontend scaffold with mock API, role-based routing, and basic status display for the SafeGAI Hybrid App.

## Steps
1. Create branch feat/m01-hybrid-app-mock from main
2. Create apps/frontend/package.json, tsconfig.json
3. Create TypeScript types matching contracts
4. Create role permission matrix
5. Create mock API adapter
6. Create UI components (StatusPanel, EventList, EquipmentStatus)
7. Create role-based pages (UserView, OperatorView, MaintainerView)
8. Create App.tsx with routing and index.tsx entry point
9. Create adapters (localAdapter interface, mockAdapter implementation)
10. Create README.md
11. Verify JSON files pass check-json.py
12. Commit
